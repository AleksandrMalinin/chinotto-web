# Chinotto Website

Landing page for Chinotto — a minimal desktop thinking tool.

Capture thoughts instantly.  
Recover context later.

---

## Development

Install dependencies:

```bash
pnpm install
```

Start the dev server:

```bash
pnpm dev
```

Build the site:

```bash
pnpm build
```

Built with React, Vite and Tailwind.
